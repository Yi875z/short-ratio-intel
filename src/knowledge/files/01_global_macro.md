# グローバル市場・相関分析マニュアル (Global Macro Logic)

## 1. 米国経済指標の解釈ルール
「良いニュースが良いニュース（Good is Good）」なのか「悪いニュースが良いニュース（Bad is Good）」なのか、FRBの政策スタンスによって解釈が逆転するため注意せよ。
* **雇用統計 (NFP) / CPI / PCE**:
  - インフレ警戒期: 数字が強い（予想上振れ）→ 金利上昇 → 株安（特にナスダック）。
  - リセッション警戒期: 数字が弱い（予想下振れ）→ 景気後退懸念 → 株安。
  - ゴルディロックス（適温）: 数字が「強すぎず弱すぎず」→ 株高。

## 2. アセットクラス間の相関 (Inter-market Correlations)
* **米国債利回り (US10Y) とグロース株**:
  - 逆相関: 10年債利回りが4.0%を超えて急騰すると、PERの高いハイテク株（SOX指数、マグニフィセント7）は売られる。
  - 日本株への波及: SOX指数下落は、東京エレクトロンやアドバンテストへの強力な売り圧力となる。
* **ドルインデックス (DXY) とコモディティ**:
  - 逆相関: ドル高は、ドル建て資産（Gold、原油）の割高感を招き、下落要因となる。
* **原油 (WTI) と期待インフレ**:
  - 原油高はインフレ再燃懸念（CPI上昇）に直結するため、FRBのタカ派化（利上げ懸念）を招く。

## 3. 恐怖指数 (VIX) の危険水域
* **VIX < 13**: 楽観（Complacency）。突然の暴落に警戒。プットの買い場。
* **VIX 13-20**: 通常運転。押し目買いが機能しやすい。
* **VIX > 30**: パニック（Capitulation）。セリングクライマックスが近く、絶好の買い場になる可能性。

## 4. セクターローテーション (Sector Rotation)
* **金利上昇局面**: 銀行（金融）、エネルギー、バリュー株選好。
* **金利低下局面**: テック、不動産、グロース株選好。
* **景気後退局面**: ディフェンシブ（ヘルスケア、生活必需品）、金（Gold）。

# 中央銀行政策と為替ポジション分析 (Central Bank & FX Flows)

本マニュアルは、日米の中央銀行（BOJ / FRB）の政策決定メカニズムと、それが引き起こす為替（USD/JPY）の非線形な変動、およびシカゴ投機筋のポジションの巻き戻しが日本株に与える破壊的なインパクトを先読みするための分析プロトコルである。

## 1. BOJ（日本銀行）の深層と市場インパクト
日銀の動向は、日本株および円相場における絶対的な「ゲームのルール（前提条件）」である。プロは現在の日銀のスタンスを「政策の正常化フェーズ」として定義し、その副作用を監視している。

### 1.1 金融政策決定会合とフォワードガイダンス
* **イベントリスク**: 年8回開催される会合の「結果発表時間（通常11:30〜13:00の間）」は、アルゴリズムが最も狂暴化する時間帯である。発表時間が遅れるほど「政策変更（サプライズ）の可能性が高い」と市場は解釈し、ボラティリティが急騰する。
* **総裁会見（15:30〜）**: 決定内容そのものより、会見での「フォワードガイダンス（将来の政策軌道に関するヒント）」が重視される。タカ派的発言は即座に「円高・先物売り」のトリガーとなる。

### 1.2 国債買い入れとテーパリングの力学
* **メカニズム**: 日銀が国債買い入れ額を減額（テーパリング）すると、日本の長期金利（10年債利回り）が上昇する。
* **市場への波及**:
  - 銀行株（TOPIXバリュー）: 利ざや改善の思惑から強力な買いシグナル（ポジティブ）。
  - 不動産株・グロース株: 資金調達コスト増の懸念から売りシグナル（ネガティブ）。

### 1.3 ETF買い入れの現在地と「見えざる圧力」
* **現状の認識**: 日銀は現在、新規のETF買い入れを事実上停止している。かつてのような「前場にTOPIXが-2%下落したら、後場に日銀砲（買い支え）が来る」というRetail（個人投資家）の期待は完全に幻想（Trap）である。
* **真のリスク**: むしろ、日銀が過去に溜め込んだ巨額のETF残高を「いつ、どのように市場に放出（売却）するか」という出口戦略が、中長期的な日本株の上値の重し（Overhang）として機能している。

## 2. FOMC vs BOJ：日米金融政策のコントラストと為替
為替（USD/JPY）のトレンドは、単なる現在の金利差ではなく、「将来の金利差の縮小・拡大ペースの『期待値』」によって形成される。

### 2.1 日米金利差と非線形な為替インパクト
* **原則**: 米国金利↑ ＋ 日本金利↓ ＝ 日米金利差拡大（強い円安・ドル高要因）
* **非線形性（Non-linear Impact）**: 金利差が特定の水準（例：心理的節目や過去の介入警戒ライン）を超えると、為替の反応は線形ではなくなる。財務省・日銀による「為替介入（Yen Intervention）」の警戒感が高まると、金利差が拡大しても円安が進行しづらくなる（上値のキャップ）。

### 2.2 ドットチャート（Dot Plot）の読み解き方
* **定義**: FRB（連邦準備制度理事会）が四半期ごとに公表する、FOMCメンバーによる将来の政策金利の見通し分布図。
* **プロの視点（Pro Intent）**:
  - 中央値（Median）の変化を見るのは当然として、プロは「分布のばらつき（タカ派・ハト派の乖離）」と「長期的なターミナルレート（最終到達地点）」の切り上げ/切り下げに注目する。
  - 市場の織り込み（FedWatchツール等）と、実際のドットチャートに「乖離（ギャップ）」がある時、強烈なトレンドが発生する。

## 3. IMM通貨先物ポジション（シカゴ投機筋の動向）
シカゴ・マーカンタイル取引所（CME）のIMM通貨先物建玉は、ヘッジファンドやCTAなど「投機筋（Non-Commercial）」の円ポジションの偏りを示す最重要データである。毎週金曜日（日本時間土曜朝）に発表される。

### 3.1 IMM建玉残高（Open Interest）の読み方
* **データラグの補正**: CFTC（米商品先物取引委員会）のデータは「火曜日時点」の建玉であり、発表までに3日間のタイムラグがある。プロは水〜金のCVDや値動きを加味して「現在の真の建玉」を推測する。

### 3.2 円ショートの極値と「巻き戻し（Short Cover）」の破壊力
* **円ショートの蓄積**: 投機筋が「円売り（Yen Short）」を歴史的極値（例：15万枚以上の売り越しなど）まで積み上げている状態は、一見すると「強い円安トレンド」に見える。
* **反転のメカニズム（Retail Trap vs Pro Intent）**:
  - 素人の誤解: 「みんな円を売っているから、まだまだ円安になる」と順張りする。
  - プロの真意: 「ポジションが限界まで傾いている。わずかな悪材料（米指標の下振れや日銀のタカ派発言）が出れば、パニック的な買い戻し（Short Cover）が発生する」と狩りの準備をする。
* **ショートスクイーズ**: 限界まで溜まった円ショートが一斉に手仕舞い（円買い戻し）されると、流動性が枯渇し、数円単位の暴れ狂う円高スパイクが発生する。

### 3.3 日経平均への波及メカニズム（円高ショックのトリガー）
1. トリガー: 米CPI下振れ 等による米金利低下。
2. 為替の崩壊: IMMでパンパンに溜まっていた円ショートが一斉に巻き戻され、フラッシュ・クラッシュ級の急激な「円高」が発生。
3. アルゴの連鎖（日経暴落）:
   - 海外勢のHFT（高頻度取引）アルゴリズムは「USD/JPYの下落」を「日経先物の売りシグナル」としてハードコーディングしている。
   - 為替の急落と同時に、日経225先物に無慈悲な大量の売り（ISアルゴ等）が浴びせられる。
4. 結論: 日経平均の暴落を先読みするには、チャートの形だけでなく「シカゴ投機筋の円ショートの溜まり具合（燃料の量）」を常に監視しなければならない。

## 4. 【実戦的統合】参謀の戦術的ルーティン
* **イベント前のポジション管理**: 日銀会合やFOMC、米雇用統計の直前に、IMMの円ショートが極端に偏っている場合、オプションの「ロング・ストラドル（Vega買い）」または「ファー・プットの買い（暴落ヘッジ）」の優位性が飛躍的に高まる。
* **タイムラグの利用**: 金曜日に「火曜時点のIMM円ショート拡大」が発表されたとしても、水〜金ですでに円高が進行していれば「すでに巻き戻しは終わっている」と判断し、安易な突っ込み売りを避ける。

# 市場の季節性とアノマリー大全 (Market Seasonality & Anomalies)

本マニュアルは、カレンダー（時間軸）に組み込まれた機関投資家・事業法人の「構造的な資金フロー」を先読みし、相場の転換点や偏りをシステムに推論させるためのプロトコルである。季節性は迷信ではなく、厳格なルールに基づく「需給の歪み」である。

## 1. 月初・月末の構造的フロー (EOM / BOM Flows)
年金基金やパッシブファンドは、月末・月初にポートフォリオの比率を元に戻す「リバランス」を機械的に実行する。

* **月末の売り（End of Month Rebalancing）**:
  - メカニズム: その月に株価が大きく上昇した場合、ポートフォリオ内の「株式比率」が目標を上回る。そのため、月末（特に最終営業日の大引けにかけて）機械的な利益確定売りが発生する。
  - 戦略的示唆: 月間を通じて強い上昇トレンドだった場合、月末最終日の高値づかみは厳禁である。
* **月初のアノマリー（初値買い）**:
  - メカニズム: 新たな月に入ると、給与天引きによる投信積立金（リテール資金）や、新たな運用予算を獲得した機関投資家の「新規マネー」が市場に流入しやすい。
  - 戦略的示唆: 月末の売りを吸収した後、毎月第1〜第2営業日は強い買い圧力が働きやすい。

## 2. 日本市場（JPX）の特有サイクル
日本の会計年度（4月始まり・3月終わり）に起因する強力な需給の偏り。

### 2.1 【3月末】期末配当取りとドレッシング買い
* **配当落ちの歪み**: 3月末は日本企業の多くが決算期末を迎える。高配当バリュー株（銀行・商社など）に「配当取り」の駆け込み買いが入りやすい。ただし、権利落ち日（月末の2営業日前）には、配当分だけ指数が機械的に下落するため、日経225先物のショートと現物のロングを組み合わせた裁定取引（配当剥がし）が活発化する。
* **ドレッシング買い（お化粧買い）**: 機関投資家が期末の運用報告書を見栄え良くするため、すでに上昇している保有銘柄をさらに買い上げる現象。月末に向けてトレンドが加速しやすい。

### 2.2 【4月】海外投資家の「新年度買い」
* **メカニズム**: 4月は海外年金基金などのニューマネーが日本株に配分されやすい歴史的特異月である。「4月の海外勢は買い越す」という強力なアノマリーがあり、統計的にも日経平均の勝率が極めて高い月となる。

### 2.3 【8月】夏枯れ相場と「お盆ショック」
* **メカニズム**: 欧米のヘッジファンド勢が長期のサマーバケーションに入り、市場の取引高（流動性）が極端に低下する。
* **Pro Intent（プロの真意）**: 流動性が枯渇した「真空地帯」では、仕掛け的な売り（CTAなどのショート）に対して買い向かう主体が不在となる。そのため、わずかな悪材料でフラッシュ・クラッシュ級の暴落（お盆ショック）が発生しやすい。VIX急騰の警戒月。

### 2.4 【12月】タックス・ロス・ハーベスティング（節税売り）
* **メカニズム**: 個人の課税年度末に向けて、利益と相殺するために「含み損を抱えた銘柄」が機械的に損切りされる現象。中小型株（マザーズ・グロース市場）が年末にかけて理不尽に売られ続ける原因となる。

## 3. 米国市場（US）の特有サイクル
グローバル市場のハブである米国のスケジュールは、日本株にも直接的な影響を与える。

### 3.1 【4月中旬】タックス・デー（納税売り）
* 米国の確定申告期限（通常4月15日付近）に向けて、納税資金を確保するための換金売りが出やすい。この日を通過すると、売り圧力が後退しアク抜け上昇しやすい。

### 3.2 自社株買いのブラックアウト期間（Blackout Period）
* **メカニズム**: 米国企業は決算発表前の数週間、インサイダー取引防止のため「自社株買い（Buyback）」を停止しなければならない。
* **市場への影響**: 米国株最大の「買い手」である事業法人のフローが消滅するため、相場が下落に弱くなる期間（サポート不在）。決算シーズン開始前はプットオプションの買い（ヘッジ）が機能しやすい。

### 3.3 トリプル・ウィッチング（米国版メジャーSQ）
* **スケジュール**: 3, 6, 9, 12月の第3金曜日。米国の株価指数先物、株価指数オプション、個別株オプションの3つが同時に満期を迎える日。
* **影響**: この週は米国のオプション・ディーラーによる激しいガンマ・ヘッジ（原資産の売買）が交錯し、論理的な理由なき乱高下が発生する。

## 4. SQ（特別清算指数）と「魔の水曜日」
日本のメジャーSQ（3, 6, 9, 12月の第2金曜日）およびマイナーSQ（その他の月の第2金曜日）に向けた特有の力学。

* **魔の水曜日**:
  - SQ週の水曜日は、海外ヘッジファンド（特にトレンドフォロワーのGSなど）が、期先限月への乗り換え（ロールオーバー）や、利益確定の反対売買を強行しやすい日である。
  - Retail Trap（素人の罠）: 「月・火と強かったから水曜も上がる」と順張りすると、水曜の午後から突然発生する強烈な反対売買アルゴに巻き込まれ、大火傷を負う。
* **SQ値の引力（Pinning）**:
  - 木曜日の大引けから金曜日の寄り付きにかけて、オプションの建玉（Open Interest）が最大化しているストライクに価格を固定しようとするマーケットメーカーの攻防が極限に達する。

## 5. 【実戦的統合】アノマリーを活用した戦略構築
プロの参謀は、これらのカレンダー情報を「テクニカル分析」や「手口分析」のフィルターとして活用する。

* **戦術例1（8月のVIX戦略）**: 8月にCVD（累積デルタ）の急低下（流動性枯渇）を検知した場合、安易な押し目買い（ショート・プット）は避け、逆に「ファー・プットの買い」で突発的なクラッシュに備える。
* **戦術例2（SQ週の逆張り）**: SQ週の水曜日に、前週から建玉を膨らませていた海外勢の手口が「反転」したのを検知した場合、それは「トレンドの継続」ではなく「SQ決済に向けた手仕舞い」であると推論し、即座にトレンドフォロー戦略を停止する。
