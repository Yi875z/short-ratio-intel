# 日経225・TOPIX オプション先物取引マニュアル
## グローバル・チーフ・インベストメント・オフィサー（CIO）による実戦ガイド

## 1. 日経225・TOPIX指数とオプションの構造理解

### 1.1 日経225とTOPIX：2つの指数の本質的役割
* **日経225**: 大型株・グロース寄り、海外投資家の主要参入口。流動性が高くトレンドが明確。
* **TOPIX**: 市場全体を反映。バリュー・中小型株の寄与度が相対的に高い。
* **NT倍率**: 日経225÷TOPIX の比率。通常は1.0～1.3の範囲で変動。1.3超はグロース相場・円安環境の典型。

### 1.2 3つのプレミアム要素
1. **本質的価値**: コールは max(原資産-権利行使, 0)、プットは max(権利行使-原資産, 0)。
2. **時間価値 (Theta)**: 時間経過とともに減衰。オプション売り手の収益源。
3. **ボラティリティ・プレミアム (Vega)**: 将来の価格変動率への期待（IV）。日経VIが30以上は超高IV（パニック局面）で売り有利。

### 1.3 Greeks指標と運用実務
* **Gamma**: Deltaの感応度。ATM付近で最大。ショート・ストラングルはGammaショート。
* **Vega**: IV感応度。IVが高い時はVegaショート（売り戦略）が優位。
* **Theta**: 時間価値減衰。満期1週間前（ATM付近）で減衰が最大化。
* **Delta**: 原資産との連動性。デルタヘッジで方向性を中立化。

## 2. JPX仕様と市場メカニズム
* **特別清算指数（SQ）**: 算出日は各限月第2金曜日の寄り付き直前。「寄り付き値」が使用されるためギャップに注意。
* **Pinning現象**: 建玉が集中した権利行使価格付近では、オプション売り手がヘッジのためSQに向けて価格を特定水準に押さえ込もうとする（磁石効果）。
* **立会いとJ-NETの乖離**: 16:30からのJ-NET（大口ブロック）の方向性が日中と異なる場合、機関投資家の本気度の表れとなる。

# 市場環境別・推奨オプション戦略リスト

## 1. IV（インプライド・ボラティリティ）が高い時
* **環境定義**: 日経VI > 25、または急落直後。
* **推奨戦略**: 「売り」主体（Vega Short）。
  - **ショート・ストラングル**: レンジ相場予想時。
  - **クレジット・スプレッド**: 方向性はあるが、IVの剥落も狙いたい時。
  - **カバード・コール**: 現物株の防衛。
* **禁止事項**: ネイキッド（裸）のオプション買い。

## 2. IVが低い時
* **環境定義**: 日経VI < 18、閑散相場。
* **推奨戦略**: 「買い」主体（Vega Long）。
  - **ロング・ストラドル/ストラングル**: イベント前（雇用統計、CPI）に、どちらかに大きく動くことを期待する場合。
  - **デビット・スプレッド**: リスク限定で方向性を狙う場合。
  - **カバード・プット**: 下落を狙うが、IV上昇による損失を防ぎたい場合。

## 3. スキュー（Skew）の歪みがある時
* **Put Skew（Far Putが異常に高い）**:
  - 戦略: Put Backspread（プットの買い増し）。暴落時の爆発的な利益を狙う。
* **Call Skew（Far Callが異常に高い）**:
  - 戦略: Call Ratio Spread（コールの売り増し）。急騰しても限定的な利益、動かなくても利益。

# 機関投資家のガンマ・エクスポージャーと流動性プロファイル (Dealer GEX)

本マニュアルは、市場全体のオプション・ディーラー（マーケットメーカー）が抱える「ガンマ（Gamma）」の偏りを可視化し、相場が「静寂（レンジ）」になるか「狂乱（トレンド/暴落）」になるかを事前に察知するための最高峰の流動性分析プロトコルである。

## 1. マーケットメーカーの行動原理（デルタ・ヘッジ）
オプション市場において、個人投資家やファンドの注文の反対側（カウンターパーティ）には、常にアムロ（ABN Amro）やソシエテ・ジェネラルなどの巨大なマーケットメーカー（ディーラー）が存在する。
* **大原則**: ディーラーは方向性（相場の上下）でギャンブルをしない。彼らは常にポートフォリオのデルタ（方向性リスク）を「ゼロ」に保とうとする（デルタ・ニュートラル）。
* **ヘッジ行動**: 顧客にオプションを売った（または買った）ディーラーは、リスクを相殺するために「原資産（日経225先物など）」を機械的に売買し続ける。この原資産の売買フローこそが、市場のボラティリティを決定づける。

## 2. ポジティブ・ガンマ環境 (+GEX) ：市場の「鎮痛剤」
市場全体のディーラーが**「ガンマ・ロング（Gamma Long）」**の状態にある環境。通常、相場が緩やかに上昇している時や、高値圏で安定している時に発生する。

### 2.1 メカニズム
* 投資家はカバード・コールなどで「コールを売り」、下落に備えて「プットを買う」。その反対側で、ディーラーは「コールを買い」「プットを売って」いる状態（＝ガンマ・ロング）。
* **ディーラーの機械的ヘッジ**:
  - 相場が**上昇**する → ディーラーのデルタがプラスになるため、先物を**「売る」**。
  - 相場が**下落**する → ディーラーのデルタがマイナスになるため、先物を**「買う」**。

### 2.2 市場への影響（Pro Intent vs Retail Trap）
* **現象**: ディーラーが「逆張り」の売買を巨額に行うため、相場の値動きが極端に重くなる。ボラティリティが抑え込まれ、特定の権利行使価格に吸い寄せられる「Pinning現象」が強力に機能する。
* **戦略的示唆**: この環境下でのブレイクアウト（上抜け・下抜け）は「ダマシ」になりやすい。Retail（素人）はトレンド発生と勘違いして飛びつくが、プロは「ショート・ストラングル」などのレンジ戦略で時間価値（Theta）を静かに搾取する。

## 3. ネガティブ・ガンマ環境 (-GEX) ：市場の「起爆剤」
市場全体のディーラーが**「ガンマ・ショート（Gamma Short）」**の状態にある環境。相場が重要なサポートラインを割り込んだり、パニック的なプット買いが殺到した時に発生する。

### 3.1 メカニズム
* 投資家が暴落ヘッジのために大量の「プットを買い」、または投機的に「コールを買う」。その反対側で、ディーラーはオプションを「売って」いる状態（＝ガンマ・ショート）。
* **ディーラーの機械的ヘッジ**:
  - 相場が**下落**する → ディーラーの損失が急拡大（デルタがプラス化）するため、損失を防ぐために先物を**「慌てて売る」**。
  - 相場が**上昇**する → ディーラーのデルタがマイナス化するため、先物を**「慌てて買う（ショートカバー）」**。

### 3.2 市場への影響
* **現象**: ディーラーが「順張り」の売買を強制されるため、下落がさらなる下落を呼ぶ「負のスパイラル（フラッシュ・クラッシュ）」が発生する。ボラティリティ（VIX/日経VI）が爆発的に上昇する。
* **戦略的示唆**: ネガティブ・ガンマ環境では、CVD（累積デルタ）に表れる海外勢の攻撃的アルゴリズム（IS等）の威力が普段の数倍に増幅される。逆張り（押し目買い）は自殺行為であり、徹底した順張り（トレンドフォロー）または「プットの買い」で暴落に乗るべきである。

## 4. ゼロ・ガンマ・レベル（Gamma Flip / Vol Trigger）
市場の性質が「ポジティブ（静寂）」から「ネガティブ（狂乱）」に切り替わる、最も重要な分水嶺（境界線）となる価格帯。
$GEX = \sum (\Gamma \times Open Interest)$ がプラスからマイナスに転じる損益分岐点である。
* **プロの視点**:
  - 現在価格が「ゼロ・ガンマ・レベル」より**上**にある場合：市場は安定しており、押し目買いが機能する。
  - 現在価格が「ゼロ・ガンマ・レベル」を**下抜けた**場合：市場の流動性が一気に消滅し、パニック売りが始まる。ここが「暴落のスタートライン」となる。

## 5. 【実戦的統合】GEXプロファイルを活用した参謀の戦略判断
| 環境 | ディーラーのヘッジ行動 | 相場の特徴 | 推奨されるオプション戦略 | 現物・先物戦略 |
| :--- | :--- | :--- | :--- | :--- |
| **+GEX (ポジティブ)** | 逆張り（高値売り/安値買い） | ボラ低下、Mean Reversion（平均回帰）、Pinning | ショート・ストラングル、クレジット・スプレッド | レンジ逆張り（VWAPでの反発狙い） |
| **-GEX (ネガティブ)** | 順張り（安値売り/高値買い） | ボラ爆発、一方向への強いトレンド、乱高下 | ロング・ストラドル、プット・バックスプレッド | 徹底した順張り（ブレイクアウト） |

* **戦略的ルーティン**: 
  1. 朝イチで日経VIの数値を確認し、建玉集中ストライク（マグネット）を特定する。
  2. 現在位置が「ガンマ・ポジティブ」か「ガンマ・ネガティブ」かを判定する。
  3. もしネガティブ・ガンマ領域に突入していれば、GPIFなどの下支え（逆張りフロー）すらもディーラーの巨大な「順張りヘッジ売り」に飲み込まれる危険性があるため、リスク許容度（2%ルール）を極限まで引き締め、買いポジションを縮小する。
